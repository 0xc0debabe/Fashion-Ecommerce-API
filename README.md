# 👔 Fashion-Ecommerce-API
패션 제품을 관리하고, 사용자들이 온라인에서 쇼핑할 수 있도록 지원하는 RESTful API입니다.  
이 API는 제품 검색, 카트 관리, 주문 처리 등 다양한 기능을 제공합니다.

## 📆 프로젝트 기간
24.10.14 ~ ?

## 🛠️ 기술 스택
- **Language** : JAVA
- **Framework** : Spring Boot
- **Build** : GRADLE
- **Database** : MySql
- **Test** : JUnit5
- **JDK** : OpenJDK-17
- **Cache** : Redis
- **Authentication** : JWT
- **Persistence & Querying** : Spring Data JPA, QueryDSL
